# node-email

## HOW TO
1. Run `npm install` to install depedencies
2. Copy .env.example to .env
3. Fill .env config file with your email credential
```
# Email Credential
EMAIL_HOST=
EMAIL_PORT=
EMAIL_SECURE=
EMAIL_USERNAME=
EMAIL_PASSWORD=
EMAIL_DEFAULT=

SEND_TO=
```
4. Run command to send email
```
npm start
```
